# Healthcare Appointment No-Show Analysis

This project analyzes factors that contribute to missed healthcare appointments using a dataset sourced from Kaggle.

## Tools Used
- Power BI
- Data Visualization
- Exploratory Data Analysis

## Key Insights
- Patients aged 19–35 showed the highest no-show rates (~24%)
- Longer scheduling delays significantly increased missed appointments
- SMS reminders alone did not appear to reduce no-show rates in this dataset
